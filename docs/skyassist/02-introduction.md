# 2. Introduction

## 2.1 Definitions, Acronyms, Abbreviations

| Abbreviation/Acronym | Definition |
|----------------------|------------|
| **Agent** | A powerful AI companion that can handle a range of interactions and tasks, from resolving issues requiring complex conversations to autonomously determining the best action to take based on its instructions and context. Agents coordinate a collection of language models, along with instructions, context, knowledge sources, topics, actions, inputs, and triggers to accomplish desired goals. |
| **Agentic AI Architecture** | AI Systems designed to resolve complex problems with limited direct human supervision. These systems are composed of multiple conversable agents that converse with each other and can be orchestrated centrally or self-organize in a decentralized manner. |
| **AI Builder** | The feature in Microsoft's Power Platform that enables users to easily add artificial intelligence capabilities to their apps and workflows using pre-built and customizable AI models, without requiring extensive coding or data science expertise. |
| **Azure AI Foundry** | Azure AI Foundry is a trusted platform that empowers developers to drive innovation and shape the future with AI in a safe, secure, and responsible way. |
| **Azure Key Vault** | A cloud service for securely storing and accessing secrets. A secret is anything that you want to tightly control access to, such as API keys, passwords, or certificates. |
| **Azure Resources** | Resources provided by Microsoft's Azure cloud platform, including Resource Groups, Key Vault, Storage account, and Application Insights. |
| **Bot** | A chatbot is a software program designed to simulate conversation with human users, providing automated responses to inquiries and engaging users through messaging platforms. |
| **CDP** | Customer Data Platform. Platform for unified customer insights and analytics. |
| **Copilot Studio** | Copilot Studio is a graphical, low-code tool for both creating an agent—including building automation with Power Automate—and extending a Microsoft 365 Copilot with your own enterprise data and scenarios. |
| **GPT** | Generative Pretrained Transformer, a type of AI model. |
| **HTTPS** | Hypertext Transfer Protocol Secure, an extension of the Hypertext Transfer Protocol. It is used for secure communication over a computer network, and is widely used on the Internet. |
| **OAuth** | An open standard for access delegation, commonly used as a way for Internet users to grant websites or applications access to their information on other websites but without giving them the passwords. |
| **OpenAI** | An artificial intelligence research organization that aims to ensure that artificial general intelligence (AGI) benefits all of humanity. The organization is known for developing advanced AI models, such as the GPT (Generative Pre-trained Transformer) series, which includes models like GPT-3 and GPT-4. |
| **Power Automate** | A Microsoft product used for automating workflows. |
| **Power Platform** | A suite of Microsoft products, including Power Apps, Power Automate, and others. |
| **RPA** | Robotic Process Automation, a technology used to automate repetitive tasks. |
| **TLS** | Transport Layer Security, a protocol that ensures privacy between communicating applications and their users on the internet. |

## 2.2 Solution Architecture Document Scope

This document will cover the following areas:

**Solution Overview**: A high-level description of the conversational bot solution, including its purpose and key functionalities.

**System Integration**: Detailed architecture on how the conversational bot will integrate with existing SkyCart enterprise systems, including order management, catalog services, and customer support systems.

**Data Management**: Outline of the data flow, storage, and management strategies to ensure data integrity, security, and compliance with relevant regulations.

**User Interaction**: Design of the user interface and experience, focusing on the interaction between the bot and SkyCart customers through various channels such as the website, mobile app, and social media platforms.
